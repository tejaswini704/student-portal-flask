# Student Management System (Flask)

## Description
A web-based Student Management System built using Flask and SQLite with role-based login and full CRUD operations.

## Features
- Login / Registration system
- Role-based dashboards (Admin, Student, Teacher)
- Add / View / Edit / Delete students
- Search functionality
- CSV export
- Session management

## Technologies Used
- Python Flask
- SQLite
- HTML, CSS

## How to Run
1. Install requirements: pip install -r requirements.txt
2. Run app.py
3. Open: http://127.0.0.1:8000
